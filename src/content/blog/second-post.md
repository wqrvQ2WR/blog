글에 이미지를 넣는 방법은 두 가지입니다.

## 1. 글쓰기 페이지에서 버튼으로

[글쓰기](/create) 페이지에서 **🖼️ 이미지 넣기** 버튼을 누르면 이미지가 업로드되고, 커서 위치에 마크다운이 자동으로 들어갑니다. 맨 위의 **대표 이미지**를 선택하면 글 상단과 목록 카드에 표시됩니다.

## 2. 마크다운으로 직접

이미지 파일을 `public/images/` 폴더에 넣고 이렇게 쓰면 됩니다.

```md
![이미지 설명](/images/파일이름.jpg)
```
